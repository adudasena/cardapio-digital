# 🍽️ Cardápio Digital - Full Stack Project

A simple and functional Digital Menu system. This project allows users to view food items and register new products in real-time.

## 🚀 Technologies

### **Backend**
* **Java 17** with **Spring Boot 3**
* **Spring Data JPA** (Database persistence)
* **PostgreSQL** (Relational Database)
* **Lombok** (Clean code)
* **Maven** (Dependency management)

### **Frontend**
* **React** with **TypeScript**
* **Vite** (Build tool)
* **TanStack Query (React Query)** (Data fetching and cache)
* **Axios** (HTTP client)
* **CSS Modules** (Styling)

---

## 📁 Project Structure

- `/backend`: REST API developed with Spring Boot.
- `/frontend`: Web interface developed with React + Vite.

---

## 🛠️ How to run the project

### 1. Backend
- Go to the folder: `cd backend`
- Configure your PostgreSQL database in `application.properties`.
- Run the command: `./mvnw spring-boot:run`
- API URL: `http://localhost:8080/food`

### 2. Frontend
- Go to the folder: `cd frontend`
- Install dependencies: `npm install`
- Start the project: `npm run dev`
- Local URL: `http://localhost:5173`
